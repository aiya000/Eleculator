# Elculator

This is an __my__ example for Electron App.  
( 'Electron-Calculator' )

## Require

- npm

## How to run

```
$ npm run build  # first only
$ npm run start
```

You need not run 'grunt' :)

## How to generate document

If you have yuidocjs.

```
$ grunt
$ npm run doc
```

You can see ./doc/index.html .
